# github-analytics-pipeline
Pipeline de dados ponta a ponta (ELT) para análise de métricas da API do GitHub. Arquitetura moderna utilizando PyAirbyte para ingestão, dbt para modelagem medallion e Dagster para orquestração. Infraestrutura provisionada via Terraform e automação completa de CI/CD via GitHub Actions com execução serverless no Google Cloud Platform (Cloud Run).
